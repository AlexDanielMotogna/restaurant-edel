# Restaurant Edel — Markenrichtlinien

Übersicht aller Farben, Schriften und Designelemente, die auf der Website restaurant-edel.at und in den Druckmenüs verwendet werden.

---

## Farbpalette (Hexcodes)

### Hauptfarben

| Farbe | Hex | Verwendung |
|-------|-----|------------|
| **Edel-Grün** (Hauptfarbe) | `#18312e` | Hauptfarbe der Marke, dunkler Hintergrund, Branding |
| **Edel-Gold** (Akzent) | `#c7a36c` | Akzente, Hervorhebungen, Preise, Schmuckelemente |
| **Dunkles Gold** (Dekoration) | `#8B6914` | Rahmen, dekorative Elemente, Trennlinien |
| **Mittleres Gold** | `#5a4510` | Untertitel auf hellem Hintergrund |

### Helle Hintergrundfarben

| Farbe | Hex | Verwendung |
|-------|-----|------------|
| **Reines Weiß** | `#ffffff` | Überschriften auf dunklem Hintergrund |

| **Sehr helle Creme** | `#f9f6f1` | Heller Hintergrund (Druckmenüs) |
| **Off-White** | `#f5efe4` | Allgemeiner Text auf dunklem Hintergrund |

### Textfarben

| Farbe | Hex | Verwendung |
|-------|-----|------------|

## Typografie

### Hauptschriften

| Schriftart | Verwendung |
|------------|------------|
| **Fraunces** (Serif) | Überschriften, Titel, Restaurantname |
| **Urbanist** (Sans-Serif) | Fließtext, UI-Elemente, Beschreibungen |

### Schriftgewichte

- **Fraunces**: 300 (Light), 400 (Regular), 600 (SemiBold), 700 (Bold)
- **Urbanist**: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold)

### Google Fonts Import

```html
<link href="https://fonts.googleapis.com/css2?family=Fraunces:ital,wght@0,300;0,400;0,600;0,700;1,300;1,400&family=Urbanist:wght@300;400;500;600&display=swap" rel="stylesheet">
```

## Designprinzipien

### Stil
- Modern, elegant, herzlich
- Reduziertes, klares Layout
- Großzügige Weißräume
- Goldene Akzente sparsam einsetzen

### Schriftgrößen (Empfehlung)
- **H1**: 32–38 px
- **H2**: 26–32 px
- **H3**: 18–24 px
- **Fließtext**: 16 px
- **Kleintext / Bildunterschriften**: 14 px
- **Metadaten / Allergene**: 12–13 px

### Buchstabenabstände
- **Überschriften (uppercase)**: 2–8 px letter-spacing
- **Subtitel**: 3–4 px letter-spacing
- **Fließtext**: 0–0,5 px letter-spacing

## Beispiel-Anwendungen

### Karte / Menü (heller Hintergrund)
- Hintergrund: `#f9f6f1` (sehr helle Creme)
- Überschriften: `#2c2c2c` oder `#18312e`
- Akzente / Preise: `#c7a36c` oder `#8B6914`

### Website / Social Media (dunkler Hintergrund)
- Hintergrund: `#18312e` (Edel-Grün)
- Überschriften: `#ffffff` (Weiß)
- Fließtext: `#e8e0d5` (Helle Creme)
- Akzente: `#c7a36c` (Gold)

### Druckmaterial / Karten
- Hintergrund: `#f9f6f1`
- Border / Rahmen: `#c7a36c` (1 px)
- Border-Radius: 8 px
